# 超级马里奥 · 迷你版

这是一个用原生 HTML/CSS/JavaScript 编写的超级马里奥风格小游戏。你可以通过方向键移动角色，空格键跳跃，收集金币并到达旗杆。

## 本地运行

### 方式一：Node 预览服务器（推荐）

```bash
npm install
npm run dev
```

默认端口为 `3000`，浏览器打开 `http://localhost:3000`。

### 方式二：Python 简易服务器

```bash
python3 -m http.server 8000
```

浏览器打开 `http://localhost:8000`。

## 操作说明

- 左右方向键 / A、D：移动
- 空格：跳跃
- 重新开始按钮：重置关卡
