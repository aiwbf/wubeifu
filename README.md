# 北府视界概念图形

一个纯前端网页程序，用 Canvas 动态展示哲学概念知识图谱。程序无需构建步骤，可直接部署到 GitHub Pages。

## 功能

- 初始概念为“自由”
- 输入任意哲学概念后生成关联图谱
- 点击任何节点会以该概念为新中心重新生成图谱
- 右侧解释包含概念定义、核心结构、理论脉络、当代人物与研究学者、争论焦点
- 支持滚轮缩放、拖动画布、拖动节点、探索历史
- 内置哲学本体数据、当代学者索引与确定性生成器，可持续向未知概念延展

## 运行

在当前目录启动静态服务器：

```bash
python -m http.server 5174
```

然后访问：

```text
http://localhost:5174
```

也可以直接部署到 GitHub Pages。由于全部资源都是相对路径，发布到仓库根目录或子目录都能运行。
