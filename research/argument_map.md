# 论证地图（Step 1）

用法：正文只能从这里“取论点”，不能临时发散造新结论。

## ARG-01 事件“再爆发”不是单点触发，而是多机制叠加

- 核心主张：爱泼斯坦事件在2018-2019年的再爆发，是“诉讼持续 + 调查报道 + 新检察行动”叠加结果。
- 证据链：E-2008-003, E-2011-001, E-2018-001, E-2019-001, E-2019-003；C-006, C-017。
- 反方观点：只是检察机构“换届效应”。
- 回应：换届可解释时点，但无法单独解释CVRA诉讼与媒体证据累积。
- 不确定性：各机制边际贡献尚未可精确量化。

## ARG-02 关键争议在“程序正义与检察裁量边界”

- 核心主张：公众争议核心并非仅是个体犯罪叙事，而是制度如何处理高敏感案件。
- 证据链：E-2007-002, E-2019-001, E-2020-003；C-003, C-004, C-005。
- 反方观点：只要后续再起诉，前期程序问题可被“结果修复”。
- 回应：程序权利是独立法益，不因后续起诉自动消失。
- 不确定性：不同巡回法院对CVRA前置适用仍有理论分歧空间。

## ARG-03 “主案终止”并未终止法律与治理过程

- 核心主张：2019主案终止后，问责转移为共犯审判、机构调查与档案公开争议。
- 证据链：E-2019-006, E-2020-001, E-2021-004, E-2023-001, E-2025-004；C-009, C-010, C-013, C-015。
- 反方观点：主案终止意味着法律层面已结束。
- 回应：后续刑事、监察、信息公开程序均在继续，且会塑造制度后果。
- 不确定性：公开材料仍无法覆盖全部调查范围。

## ARG-04 媒体与信息生态对制度信任有“双刃剑效应”

- 核心主张：调查报道提升问责可见度，但信息真空期会被阴谋叙事占据。
- 证据链：E-2018-001, E-2023-001；C-018, C-019, C-020；S28,S29。
- 反方观点：媒体只会“放大噪音”。
- 回应：无媒体介入时，程序争议往往不易进入公共审视议程。
- 不确定性：需后续补充平台数据验证扩散强度。

## ARG-05 未来政治影响存在两条相反路径（A/B）

- Scenario A（程序改革主导）：
- 触发条件：解封争议进入规则化、检察协商透明度改革落地、跨辖区移送可追踪。
- 影响路径：司法公信力边际修复 -> 阴谋叙事回落 -> 政策竞争转向制度细节。
- 风险代价：公开过度可能侵害隐私，改革执行成本高。
- 观察指标（1-3年）：立法进度、司法规则修订、案件沟通规范、民调中的司法信任。

- Scenario B（阴谋政治主导）：
- 触发条件：高关注节点持续缺乏高质量解释、平台放大未证实叙事、党派竞争激励极化。
- 影响路径：程序性议题被“阵营化” -> 证据标准下沉 -> 制度信任进一步分裂。
- 风险代价：正当程序被挤压、隐私伤害与误伤概率上升。
- 观察指标（3-10年）：极化指数、阴谋信念调查、司法机构信任差距、立法极化度。

- 若出现可核验数据否定“信息真空与谣言扩散”关系，则 `ARG-04` 需降级为备选解释。
